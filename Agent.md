# 项目 Agent 总说明

## 比赛基本信息

- **比赛名称**：第十二届统计建模相关竞赛项目
- **比赛主题**：服务国家战略，创新统计赋能
- **项目选题方向**：面向新型城镇化与区域协调发展，利用遥感数据、空间统计与社会经济数据识别粤北非珠三角四线城市住宅空置风险，并分析其空间扩散机制与未来转移概率。
- **当前研究对象**：广东省韶关市，后续可根据数据可得性扩展至粤北其他非珠三角城市。
- **核心空间单元**：500m × 500m 标准地理格网，作为遥感特征、市场特征、空间统计模型和状态转移模型的统一承载单元。
- **核心被解释对象**：住宅空置风险复合指数，暂命名为 `RVRI`，用于表征“住房存量较高但人类活动活力不足”的空间单元风险水平。

## 研究问题

1. **识别问题**：如何利用遥感、建成环境、市场活力和社会经济变量构建高维度的“住宅空置风险复合指数”？
2. **区分问题**：如何区分“老城衰退型空置”和“新区扩张型空置”两类机制不同的空置风险？
3. **预测问题**：空置风险是否具有空间自相关和状态转移规律？未来某一期哪些区域更可能从“低风险”转为“高风险”？

## 总体技术路线

本项目采用“多源数据融合 - 风险指数构建 - 空间类型识别 - 状态转移预测”的递进式建模框架：

1. **Q1：住宅空置风险识别**
   - 建立统一格网底座。
   - 融合 Sentinel-2、NPP-VIIRS 夜间灯光、行政边界、市场挂牌或带看数据等多源信息。
   - 构建住宅空置风险复合指数 `RVRI`，为后续分类与预测提供核心因变量。

2. **Q2：空置风险类型区分**
   - 基于 `RVRI`、建成区扩张、老城建成年代、人口/产业活力、空间位置等变量，区分“老城衰退型空置”和“新区扩张型空置”。
   - 重点解释不同空置类型背后的形成机制，而不只给出风险高低排序。

3. **Q3：空间扩散与转移预测**
   - 检验 `RVRI` 的全局与局部空间自相关。
   - 构建空间权重矩阵，识别高风险集聚区、扩散方向与邻近效应。
   - 使用状态转移矩阵、空间 Markov 链或相关时空模型，预测未来低风险格网转为高风险格网的概率。

## 建模交付要求

- **科学性**：指标定义需要与“住宅空置”的经济含义一致，即同时体现住房存量、居住活力、市场活跃度和空间位置条件。
- **可解释性**：所有指数方向必须清晰，例如夜间灯光越弱通常代表居住活力越低，建筑强度越高通常代表潜在住房存量越高。
- **可复现性**：Q1、Q2、Q3 的核心数据处理和模型计算应保存为脚本或 notebook，并记录输入数据、输出数据和关键参数。
- **空间一致性**：所有空间数据需要统一坐标系、统一空间单元，并避免经纬度单位与米制单位混用造成面积或距离误差。
- **政策相关性**：模型结论应服务于新型城镇化、存量住房盘活、城市更新、区域协调发展和“十五五”时期资源优化配置。

## 当前目录分工

- `Q3/`：空间自相关、空间扩散机制和未来状态转移概率预测。
- `Paper/`：论文正文、图表、附录和竞赛提交材料。
