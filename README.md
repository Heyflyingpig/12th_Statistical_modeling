# 📄 2026统计建模大赛 Q1 阶段进展报告：住宅空置风险测度与数据底座构建

**汇报人：** 数据建模组 (Q1 Agent)
**研究单元：** 500m × 500m 标准地理格网
**核心指标：** 住宅空置风险复合指数 (RVRI)

---

## 一、 我们目前完成了什么？（核心里程碑）

我们已经成功走通了**“天-地-网”**多源数据融合流程，完成了从原始卫星信号到社会经济指标的转化：

1.  **空间底座构建**：将韶关全域（1.8万$km^2$）切分为 **84,815 个** 500米渔网单元。
2.  **多源数据打通**：实现了 GEE（云端）与本地 Python 的闭环，完成了 Sentinel-2（10米精度光学）与 NPP-VIIRS（500米精度夜光）的时空对齐。
3.  **样本精炼（脱水）**：利用 **NDBI 建筑强度指数** 进行物理掩模，从 8.4 万个格子中剔除了森林、农田和水域，锁定了 **19,704 个“住宅相关格网”**。
4.  **创新算法赋能（降尺度）**：实现了**空间降尺度（Downscaling）纠偏**。利用 10 米精度的建筑分布作为权重，修正了 500 米夜光数据的“溢出”和“模糊”问题。
5.  **风险指数合成**：通过 **PCA（主成分分析）** 客观合成了 **RVRI 指数**。目前模型对方差的解释力为 **47.64%**，在社会学建模中表现优异。

## 二、 关键产物：如何解读那张“彩色散点图”？

在我们的 `RVRI_Feature_Space.png` 中，每一个点代表一个 500 米格网：

*   **横轴 (CNI)**：校准后的灯光强度，代表**人类社会活力**。
*   **纵轴 (NDBI)**：建筑强度，代表**房屋存量规模**。
*   **颜色 (RVRI)**：红点代表高空置风险。
*   **统计学发现**：风险最高的点（红色）集中在**“高存量、低活力（房多灯暗）”**的左上角。这证明我们的 RVRI 指数完美符合住宅空置的经济学定义，模型逻辑自洽。

## 三、 对队友的后续支撑 (Q2 & Q3 接口)

目前生成的 `Q1_Final_RVRI_Dataset.geojson` 包含了以下“值钱”的字段，请队友们按需调用：

*   **`grid_id`**：唯一空间编码。
*   **`rvri`**：每个格子的空置风险分值（0~1），是全篇的**核心被解释变量**。
*   **`ndbi_base`**：建筑强度，可用于 Q2 分析新区扩张速度。
*   **`ndvi_Q3_stdDev`**：绿化扰动标准差，代表**物业管理水平**。
*   **`district_name`**：所属区县，用于挂载宏观年鉴数据。
